# 🚀 Protótipo Lava Car

[![Status](https://img.shields.io/badge/Status-Conclu%C3%ADdo-brightgreen)](https://github.com/rita-cassia-developer/prototipo-lavacar)

## 📝 Descrição do Projeto

Este projeto é um **protótipo de website para um lava car**, desenvolvido com foco em **Clean Code**, HTML, CSS, responsividade, acessibilidade e boas práticas de desenvolvimento. O objetivo foi criar uma interface profissional e funcional para um lava car, demonstrando a aplicação de princípios de código limpo e design moderno.

## ✨ Funcionalidades

- **Design Responsivo:** Layout adaptável para diferentes dispositivos (desktops, tablets e smartphones).
- **Clean Code:** Código limpo, organizado e de fácil manutenção.
- **Estrutura Semântica:** Utilização de HTML5 para uma organização clara e acessível do conteúdo.
- **Acessibilidade (WCAG):** Implementação de práticas para garantir que o site seja utilizável por todos.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação do conteúdo e semântica da página.
- **CSS3:** Estilização e responsividade do layout.
- **Clean Code:** Aplicação de princípios de código limpo para garantir a legibilidade, manutenção e escalabilidade do projeto.
